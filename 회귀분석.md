회귀분석
===
상관관계 (상관계수 확인) 
--


상관계수에 대한 t-test 
--
> * 귀무가설 : 두 변수 간에 상관관계가 없다.
> * 대립가설 : 두 변수 간에 상관관계가 있다. 

```python
from scipy import stats
stats.pearsonr(x, y) # 피어슨 계수
stats.spearmanr(x, y) # 스피어만 계수
stats.kendalltau(x, y) # 켄달 계수 (켄달타우)
```
